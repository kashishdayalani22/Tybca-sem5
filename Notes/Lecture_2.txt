=================
difference between constructor and method

1. constructor is created to initialize and declare instance variable while method is used to perform business logic.
2. constructor is called only once per object while we can call method any number of times per object.
3.method name can be anything  but constructor name should be __init__()
4.method will be executed if we call while constructoe will be executed automatically ,if we are creating object.
